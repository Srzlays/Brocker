## Sobre el Dashboard
Esta es una aproximación simple a un Brocker financiero desarrollado en lenguaje de programación python para consolola de Linux y Windows.

## Expecificaiones
El Dashboard fue desarrollado en la version 3.14.4 de python. Y hace uso de las librerías:
pandas, yfinance, altair y streamlit. Se recomienda instalar un entorno virtual con python 3.14.4 e instalar las siguientes librerías:

```bash
pip3 install pandas
pip3 install yfinance
pip3 install streamlit
pip3 install altair
```

## Compilacion y Ejecución 
1. En distribuciones linux Ubuntu y Debian:
Se abre la consola de linux
```bash
ctrl + alt + t
```
Y se ejecuta el dashaboard
```bash
streamlit run Dashboard.py
```

2. En las versiones 10 y 11 Windows usando Visual Studio Code 
Abre la carpeta del proyecto con Visual Studio Code y en la terminal ejecuta el archivo Dashboard.py con Streamlit:
```bash
streamlit run .\Dashboard.py
```
